# PassLocker

1. Clonar proyecto
2. ```npm install```
3. Clonar el archivo ```.env.template``` y renombrar a ```.env```
4. Cambiar las variables de entorno
5. Levantar la BBDD (detached/desacoplado)
```
docker-compose up -d
```

6. Levantar: ```npm run start:dev```
